Slide show with Polymer

HTML experiments
Presentation
This experiment has been built with Polymer library.
You can control the slide show with:
  • Left click, Right or Down arrow and Enter key to move to the next item
  • Left or Up arrow to move to the previous item
  • 'g' to display a 'Go to slide' dialog box
  • 'm' to display slides grid and click on a slide to go to it
  • 'n' to display a navbar at the bottom of the screen
  • Adding CSS class preventClickToNext to an element will disable go to next item for the click event on it
Presenter mode
Presenter mode can be started by pressing 'p' key anytime. Another window is opened on the same slide as the current one. This second window is controlled by the main window. No control is enabled in this client window. Communication is done thanks to postMessage method.
Animated items
You can add animated items with animated-list polymer element:
  • li tags are displayed sequentially with fade-in effect
  • Fade-out effect is also applied when going back in the slide show

    Slide code declaration
    A general slide is declared in HTML by adding these code: 
    	<section>
		<div>
			<header>Title of the slide</header>
			<content>
				Content of the slide
			</content>
			<footer>Footer of the slide if any</footer>
		</div>
	</section>
    But you can also add your own style for specific slides :)
